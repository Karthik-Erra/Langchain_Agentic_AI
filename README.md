# LangGraph Chatbot Application

This project is a **general-purpose chatbot application** built using **LangGraph** with a **modular code structure**.  
The application provides a conversational interface and can be easily extended with additional tools, APIs, or workflows.

## Features

- Built using **LangGraph**
- **Modular architecture** for easy maintenance and extension
- Simple **Streamlit-based UI**
- Easy to run locally
- Designed to integrate tools, APIs, and LLM workflows

## Project Structure

project/
│
├── app.py # Main Streamlit application
├── graph/ # LangGraph workflow logic
├── utils/ # Helper utilities
├── tools/ # Tool integrations (APIs, functions, etc.)
├── config/ # Configuration files
└── requirements.txt # Python dependencies


- **Create Virtual environment**
- **pip install -r requirements.txt**
- **streamlit run app.py**
